# FluxPath MMU Platform

This directory was generated by `bootstrap_fluxpath.sh`.

## Layout

- `install.sh` – main FluxPath installer (whiptail UI, advanced MMU wizard)
- `config/fluxpath_config.json` – MMU configuration (created by wizard)
- `ui/` – Mainsail/Fluidd panel JSON (generated by installer)
- `scripts/` – helper tools (status monitor, config viewer, placeholders)
- `klipper/fluxpath_mmu_macros.cfg` – Klipper MMU macro + shell-command bridge

## Basic usage

```bash
cd ~/FluxPath
./install.sh
